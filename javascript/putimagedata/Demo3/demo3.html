Simple Demo that show how to display images from Raster Master javascript export function.
This demo shows more advanced use imageData(), putImageData(), and drawImage()
We are converting the imagaData to a Data URL so it can also be used with drawImage()
drawImage has more advanced options for displaying images. In this example we can see
that drawImage() displays trnsparent parts of the image correctly over other colors
putImageData only displays it correcly only over the canvas background color
